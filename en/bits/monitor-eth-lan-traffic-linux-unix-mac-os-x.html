Mac OS X, Ubuntu Linux, Network traffic monitor

Written by Guillermo Garron .
Date: 2013-01-30 18:07:13 -0500

There are a lot of times when you need to monitor the network traffic of a server, if you are using a *nix server like Linux or Mac OS X then you can use vnstat.

To install it on Ubuntu:

sudo apt-get install vnstat

For Mac

brew install vnstat

Measure the traffic daily and monthly

vnstat

The output will be::

Linux exclusive - vnstati

If you are working with Linux, you can also graph the info from vnstat, with vnstati. To install it on Ubuntu.

sudo apt-get install vnstati

Here some examples of the vnstati command, and the output of them:

vnstati -s -i eth0 -o /tmp/vnstat-s.png

The output is something like this:

vnstati -m -i eth0 -o /tmp/vnstat-m.png

The output is something like this:

vnstati -h -hs -o /tmp/vnstat_hs.png
