Brackets Text Editor

Written by Guillermo Garron .
Date: 2015-01-11 20:41:00 -0500

As most of you alreay know, this blog is powered by Jekyll, and therefore it is a bunch of plain text files, with Markdown syntax. In all this years I have used different text editor to write this blog.

  • Vim
  • Emacs
  • iA Writer
  • Uber Writer
  • ByWord
  • MultiMarkdown Composer
  • Erato

Most of the time it is just Vim, and Emacs is great because it highlights Markdown syntax, but lately I am using more and more brackets

Brackets is a text editor with a really light code that works great with html, css, js files and of course it works with Markdown files too, I started using it because I am learning Wordpress themes development.

It is available on Windows, Mac OS X and Linux and it is Open Source which is great for me, because that way I can use the same tool to work on my Ubuntu 14.04 or my Mac OS X laptops.

A nice feature is the possibility to add functions via extensions, you can write your own, or install one of the community ones. Brackets Git is a good one, you will find a lot of other ones here.
