Como actualizar debian Lenny a Sqeeze o 5.0 a 6.0

Escrito por: Guillermo Garron .
Fecha: 2012-06-24 08:29:00 -0400

Lo primero que tenemos que hacer antes de actualizar nuestro Debian, es realizar copias de seguridad de todo lo que nos sea importante. Vale decir nuestros datos y los archivos de configuración. Si puedes hacer una imagen de todo el servidor, mucho mejor así será muy fácil volver atrás en caso de una falla mayor (Algo que no debería suceder con Debian estable).

El segundo paso en esta actualización, es leer las notas de publicación de Debian Sqeeze.

Ahora si podemos proceder a la actualización de Debian Lenny a Squeeze.

Editamos el archivo sources.list, debes hacerlo como usuario root, o usando sudo

vim /etc/apt/sources.list

Ahora cambiamos las palabras lenny a sqeeze en todos los lugares donde estas aparezcan.

Finalmete, procedemos a la actualización en si. Ya hiciste copias de seguridad de todo verdad?

sudo apt-get update

sudo apt-get install apt dpkg apt-get

sudo apt-get dist-upgrade

Ya esta, recuerda que el proceso completo puede durar un buen tiempo, dependiendo de tu microprocesador, de tu velocidad de Internet, y de la cantidad de paquetes que tengas instalados.

Cuando termine todo el proceso, disfruta de tu Debian Squeeze.
