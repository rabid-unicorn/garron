UberWriter: El mejor editor Markdown para Ubuntu

Escrito por: Guillermo Garron .
Fecha: 2013-04-25 16:30:35 -0400

Hace algún tiempo que comencé a escribir mi blog utilizando MarkDown en lugar de algún editor visual. En realidad, ya antes de utilizar MarkDown, yo escribía los artículos en html directamente. Puesto que usaba Drupal como plataforma del sitio, me era fácil hacer esto.

Descubrí markdown, cuando experimente con Movable Type. Luego incorpore un modulo a Drupal, para escribir mis artículos en markdown.

Finalmente con el tiempo, volví a los 90s, pues comencé a publicar este sitio en html plano, sin utilizar PHP. Me mude para Jekyll, que es lo que uso para generar este sitio a partir de archivos de texto plano, escritos utilizando markdown.

Dado que utilizo Ubuntu bastante (ademas de Arch Linux y Debian), me puse a buscar el mejor editor de markdown para Ubuntu.

Fue así que descubrí UberWriter, que para mi, es el mejor editor de markdown disponible para Ubuntu en el momento.

Este editor está inspired en el iA Writer para Mac. Lo mejor de ellos, es la claridad de la pantalla que se presenta al usuario.

No hay, menús, no hay barras de opciones, no hay nada, solo el texto en la pantalla. Esto hace que escribir sea de lo más ameno, pues no hay distracciones, y el escritor puede concentrarse en lo que escribe únicamente.

UberWriter, tiene un modo llamado “focus-mode” el cual, baja la intensidad de los párrafos que ya concluiste, y centra en la pantalla, a la vez que da énfasis en el párrafo en el que estas en el momento.

Un ejemplo de ello puedes verlo en la imagen abajo.

Instalación

Ahora que ya te entro la curiosidad de provarlo, pues te muestro como se instala.

sudo add-apt-repository ppa:w-vollprecht/ppa
sudo apt-get update
sudo apt-get install uberwriter

Con eso estas, ahora solo te queda ponerte a escribir.
