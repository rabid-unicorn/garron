Dirección IP Secundaria

Escrito por: Guillermo Garron .
Fecha: 2015-01-24 06:50:00 -0500

Agregando una segunda IP a la interfaz en Linux

Agrega una dirección IP secundaria temporal

Esta será una segunda IP en la interfaz de red, que tan solo estará activan mientras no reinicies la máquina, por lo tanto es temporal.

Usando ifconfig

ifconfig [nic]:0 [IP-Address] netmask [mask] up

Y aquí un ejemplo con valores reales.

ifconfig eth0:0 192.168.1.2 netmask 255.255.255.0 up

Por si acaso, requieres ser root o permisos de root para ejecutar los comandos de arriba.

Usando el comando ip

ip address add [ip]/[mask-digits] dev [nic]

Y de nuevo un ejemplo real:

ip address add 192.168.99.37/24 dev eth0

Agregar una dirección IP secundaria permanente

Ubuntu

Para usuarios de Ubuntu el asunto sería así:

vim /etc/network/interfaces

Y agregas algo así para la segunda dirección IP

auto [NIC]:[n]
iface [NIC]:[n] inet static
address [ip.add.rr.ss]
gateway [gw.ip.ad.rs]
netmask [ne.tm.as.kk]

Y aquí tenemos un ejemplo:

auto eth0:1
iface eth0:1 inet static
address 192.168.0.1
gateway 192.168.0.254
netmask 255.255.255.0

Listo, puedes agregar tantos bloques como el de arriba, como gustes, solo ten cuidado de cambiar eth0:1 por, eth0:2, eth0:3 …. eth0:n

Igualmente si estas usando eth1, en1 o cualquier otro nombre para NIC principal, pues asegurate tambien de cambiar ello, y quedaría algo así: en1:1, en1:2 … y así.
