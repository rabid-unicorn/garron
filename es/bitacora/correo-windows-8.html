Aplicación de correo de Windows 8.1

Escrito por: Guillermo Garron .
Fecha: 2014-01-22 09:12:35 -0500
Estoy después de mucho tiempo utilizando Windows nuevamente, ahora estoy con 8.1 luego de unos dos meses en Windows 8. La versión que estoy utilizando es la PRO, pues es la que me instalaron en la laptop del trabajo. A diferencia de las versiones anteriores de Windows que venían prácticamente vacías, esta viene con algunas aplicaciones nativas. Hoy les contare un poco sobre su aplicación de correo electrónico. **Interfaz y uso** La interfaz de esta aplicación recuerda mucho la de las aplicaciones de celulares o las famosas "web apps". Posee una interfaz limpia, suave, con agradables colores y con muy pocas distracciones a la vista. Usa un modo de pantalla completa, tal como las aplicaciones de tabletas, lo que por lo menos para mi, le da un toque agradable a la hora de trabajar con ella. La interfaz es muy intuitiva, la mayoría de los controles están escondidos pero aparecen con solo hacer click derecho, y si ya utilizaste Windows 8 sabrás que llevando el ratón a la esquina superior derecha, tendrás los menús de configuración. Básicamente tienes tres paneles en la pantalla, el primero es donde cambias entre carpetas, contactos favoritos, correos marcados para seguimiento y atajos a carpetas que hubieras configurado. El segundo panel muestra la lista de correos en la carpeta actual y el tercer panel nos muestra el contenido del correo actualmente seleccionado. En la esquina superior derecha hay tres botones, uno es para un menú donde puedes responder, responder a todos y reenviar el correo, el otro es para componer un nuevo correo y el tercero es para borrar el correo actualmente visualizado **Características** Las tareas básicas están todas incluidas, se puede fácilmente responder, reenviar, borrar, marcar con no leído o leído y para seguimiento cualquier correo en la vista actual. Por otro lado, es sencillo añadir, borrar o vaciar una carpeta, y se puede también agregar un atajo a una carpeta o sub-carpeta al panel izquierdo. Lo que aún no encontré como hacer, es crear carpetas de búsqueda o filtrar correos (sin usar la búsqueda). Tampoco he podido crear reglas para el correo entrante, pero la verdad es algo que extraño poco o nada. **Tipos de cuenta soportados** La aplicación de correo nativa de Windows 8 y 8.1 soporta por defecto cuentas POP3, IMAP y ActiveSync, por lo que todos los principales proveedores de correo pueden se utilizados. - Outlook (Hotmail) - Exchange - Google (Gmail) - Yahoo email - Y cualquier otra con soporte POP3 o IMAP Como puede verse, esta aplicación es casi tan completa como la aplicación de correo de Mac. Además un punto donde supera a Outlook, es que puedes sincronizarlo con dos cuentas exchange y utilizando ActiveSync en ambas, mientras que Outlook debe utilizar IMAP con una de ellas. Yo actualmente la utilizo con mis cuentas personales, Gmail y Office 365 y mi cuenta del trabajo, un servidor Exchange. **Consideraciones finales** Luego de utilizar la aplicación por unos meses, estoy bastante contento con la misma, en realidad cada vez utilizo menos Outlook y más la aplicación nativa de Windows 8.1 para leer mi correo electrónico, tanto personal como corporativo.
